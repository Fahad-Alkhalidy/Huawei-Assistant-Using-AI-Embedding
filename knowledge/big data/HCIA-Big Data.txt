HCIA-Big Data

Type

Cloud Platform & Cloud Services (Big Data)

Capabilities to be Certified

(1) Master the technical principles and architectures of common and important big data components, including HDFS, HBase, Hive, ClickHouse, MapReduce, YARN, Spark, Flink, Flume, Kafka, ElasticSearch and ZooKeeper, and capable of using Huawei big data platform MRS;  
(2) Able to operate and develop services based on Huawei MRS;  
(3) Competent for positions related to big data development engineers.

Knowledge to Learn

Technical principles and architecture of common and important big data components. Knowledge and skills required for big data pre-sales, big data project management, and big data development.

ObjectivesMaster the technical principles and architectures of key big data components such as HDFS, HBase, Hive, ClickHouse, MapReduce, YARN, Spark, Flink, Flume, Kafka, ElasticSearch, and ZooKeeper.Operate and develop services using Huawei MRS.Be qualified for big data development roles.Understand essential knowledge and skills for pre-sales, project management, and development in big data.

Target AudienceThose who desire to become big data engineersThose who desire to obtain the HCIA-Big Data certificationJunior big data engineers

PrerequisitesBasic knowledge of network technologyFamiliarity with basic Linux operations

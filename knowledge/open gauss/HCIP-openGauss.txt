HCIP-openGauss

Type

Basic Software & Hardware (OpenGauss)

Capabilities to be Certified

Be able to compile complex SQL statements, develop APIs, manage database systems and nodes, and design and manage databases.

Knowledge to Learn

openGauss product architectures, object permission management, routine database management and check, API development, database migration, and database security management.

ObjectivesIndependently complete secondary development of a database.Perform O&M on a database system and its nodes.Perform database security management.Complete the database migration.Write complex SQL statements and develop APIs.

Target AudienceThose aspiring to become database professionalsThose aspiring to obtain the HCIP-openGauss V1.0 certification

PrerequisitesBasic knowledge of databasesBasic knowledge of Linux OSBasic knowledge of shell commandsSkill in compiling SQL statements

Training Content

HCIP-openGauss V1.0 Training

openGauss ArchitecturesSystem architectureSoftware architectureLogical architectureThread architecture

Database Object ManagementSystem catalogs and system viewsDatabase object permission managementScheduled jobs

Database System ManagementRoutine checkChecking OS and openGauss healthChecking database performanceChecking and clearing logsMaintaining tables and reindexingExporting and viewing WDRs

Advanced SQLAdvanced syntaxStored procedures and triggersApplication development

Database Migration ManagementData migration overviewopenGauss migration toolsSyntax differences between heterogeneous databasesMySQL data migration practice

Database Security ManagementClient access authenticationUsers and rolesDatabase auditEncrypted equality queryLedger databaseTransparent data encryption (TDE)

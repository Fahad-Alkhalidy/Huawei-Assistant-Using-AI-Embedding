HCIA-5G-RNP&RNO

Type

ICT Infrastructure (5G)

Capabilities to be Certified

Have the capability to perform independently completing site survey, simple planning, testing, basic optimization, and performance monitoring for Huawei 5G radio networks.

Knowledge to Learn

5G air interface principle, 5G protocol and signaling analysis, 5G radio network planning, features related to Huawei 5G radio network optimization, 5G radio network performance management and optimization, 5G radio network problem analysis and handling, and 5G typical industry applications.

Objectives

On completion of this program, the participants will be able to:Describe 5G radio network principles and functionsDescribe the basic functions of 5G base stationDescribe the General Target of applying the RRM FeaturesUnderstand the Principles of the Basic Radio Network Functions and FeaturesUnderstand the objectives and basic planning process of 5G network planning at the early stageMaster the calculation rules and parameters of the 5G radio network coverage link budgetUnderstand the link budget for NR FDD wireless network coverage and the differences between NR FDD and NR TDDHave a good command of 5G wireless network capacity estimation and planningUnderstand the site solutionsUnderstand the design principles of the 5G antenna system in different regionsDescribe how to use GENEX Probe to perform NR drive testDescribe how to configure different test plansDescribe how to collect test data by using the GENEX ProbeList basic test parameters in NR technologyDescribe the functions of the GENEX AssistantUse the GENEX Assistant to analyze DT logfilesBe able to prepare common DT analysis reportsUnderstand the function test methods of the network test toolMaster the single-site verification test indicators and requirementsUnderstand the 5G NSA&SA access process and key pointsBe able to analyze and locate typical faults encountered during the testUnderstand the definition of 5G coverageUnderstand the analysis process of 5G coverage problemsUnderstand the key factors that affect the coverage of the 5GUnderstand the solutions to 5G coverage problemsDescribe 5G radio network KPI and its definitionCollect and define 5G radio network KPI in OSS system

Target AudiencePeople who plan to participate in the HCIA-5G-RNP&RNO certificationPeople who intend to learn basic concepts and principles of 5G, and basic 5G radio network planning and optimization technology

PrerequisitesMaster basic knowledge about one type of mobile communication networks (2G/3G/4G)

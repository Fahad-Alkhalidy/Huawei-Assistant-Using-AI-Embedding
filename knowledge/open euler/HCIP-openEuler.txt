HCIP-openEuler

Type

Basic Software & Hardware (OpenEuler)

Capabilities to be Certified

The HCIP-openEuler V1.0 certification proves that you have mastered the capabilities of LAMP/LNMP application cluster deployment, Ansible- or Salt-based automated O&M, and system monitoring, and are competent for positions such as senior Linux O&M engineers, senior Linux management engineers, and enterprise application architects.

Knowledge to Learn

openEuler Service Management, openEuler Cluster Architecture, openEuler Storage Management, Automation Management, Advanced Shell Scripts, openEuler OS Security Hardening, openEuler System Monitoring, Comprehensive Practices

ObjectivesMaster the deployment of LAMP/LNMP application clusters.Use Ansible or SaltStack for automated O&M.Perform advanced openEuler system monitoring and troubleshooting.Implement openEuler OS security hardening.Develop and manage advanced shell scripts.Manage openEuler clusters and storage systems.Apply knowledge through hands-on comprehensive projects.

(2) Participate and pass any exam of the same or higher level certification from the same technology track of the current certification.

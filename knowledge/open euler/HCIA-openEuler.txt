HCIA-openEuler

Type

Basic Software & Hardware (OpenEuler)

Capabilities to be Certified

Have basic operation and management capabilities for core operating systems in enterprise data centers and be competent for Linux system O&M engineers.

Knowledge to Learn

openEuler user management, network management, permission management, storage management, software management, service management, shell basic knowledge, and Samba server setup and use.

ObjectivesUnderstand the history of operating system development.Be familiar with basic openEuler commands, such as directory operations and file viewing.Be familiar with vi and vim text editors.Have a good command of openEuler user, user group, and permission management.Have a good command of the installation of openEuler operating system application software.Have a good command of openEuler storage space management.Master the planning tasks, network, and service management of the openEuler system.Familiar with shell script basics.

Target Audience

People who are interested in operating systems.

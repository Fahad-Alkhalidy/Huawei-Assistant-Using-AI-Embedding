HCIP-Data Center Network

Type

ICT Infrastructure (Datacom)

Capabilities to be Certified

Passing the HCIP-Data Center Network certification will prove that you have solid theoretical knowledge of data center network solutions and are able to use Huawei data center network products and solutions to plan, construct, maintain, and optimize data center networks. You are competent for the position related to the data center network scenario. (account manager, project manager, pre-sales engineer, after-sales engineer, O&M engineer, etc.)

Knowledge to Learn

Data center network technical knowledge, Huawei CloudFabric solution, data center network planning and deployment, and data center network O&M.
